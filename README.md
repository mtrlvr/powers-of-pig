# Powers of Pig

A mobile-first 2048 clone where numbers are replaced by increasingly absurd pigs. Merge matching pigs to discover all 17 tiers, from tiny **Pip** to the legendary **THE LION PIG**.

**[Play now at powersofpig.com](https://powersofpig.com)**

## How to Play

- **Swipe** (mobile) or use **arrow keys** (desktop) to move all tiles
- When two matching pigs collide, they **merge** into the next tier
- Keep merging to discover all 17 pig evolutions
- Game ends when the board fills up with no valid moves

## Features

- Smooth tile sliding animations
- 17 unique pig sounds
- Haptic feedback on supported devices
- Tutorial for first-time players
- "Stuck?" help system
- Share your score with friends
- English/French language support
- Works offline (no account required)

## Tech Stack

- Vanilla HTML/CSS/JavaScript (no frameworks)
- Web Audio API for sounds
- CSS transforms for animations
- localStorage for game state
